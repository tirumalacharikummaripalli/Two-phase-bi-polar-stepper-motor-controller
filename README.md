# Two-Phase-Bi-Polar-Stepper-Motor-Controller
The Two Phase Bipolar Stepper Motor Controller is a motor controller circuit designed to drive and control bipolar stepper motor with high precision. This project uses the L297 stepper motor controller IC and the L298N H-bridge motor driver IC. The controller operates with a 12V motor power supply and supports both full-step and half-step modes of operation. Control inputs such as clock, enable and reset allow easy interfacing with microcontrollers. 
Diodes are connnected across the motor driver outputs to supress voltage splikes generated by the inductive motor windings. Resistors provide current regulation and protect the motor from excessive current. Power supply filtering capacitors improve stability and reduce electrical noise.
This controller is suitable for applications such as CNC machines, robotic systems, 3D printers, where accurate positioning and controlled motion is required.
